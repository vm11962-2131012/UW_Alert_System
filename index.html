About the Map

This interactive map is designed to help students and faculty, residents, and visitors stay informed about crime incidents in the University District. Using real-time data from the Seattle Open Data Portal, this map provides recent police and crime reports, citizen-reported incidents, and EMS calls.

Key Features

Live Crime Updates

A live-updating map of all police reports, crime reports, EMS calls or citizen-reported incidents

Rerouting for Safe Navigation

Rerouting function for users to their destinations on campus, avoiding any active crime scenes (buffers around certain areas), with adjustable travel methods

Navigation Tool

The navigation tool allows users to choose points on the map to create a safe route.

Instructions:

1. Click the navigation button in the upper right corner.

2. Click "Okay" on the popup.

3. Pick a starting point on the map; the point will be green.

4. Pick a destination point on the map; the point will be red.

5. The route directions and trip duration will display in a panel on the right side.

⚠️ Warning: If the route navigates through a high-risk area, a warning will display on the screen.
