UW Alert Mapping System

Real-Time Crime and Emergency Reporting Web Map

Project Description

The UW Alert Mapping System is a dynamic, real-time crime and emergency reporting web map designed to enhance public safety awareness around the University of Washington (UW) campus.

  • Live emergency 911 call reports
  • Historical crime data analysis
  • Campus infrastructure and navigation details

Main Functions & Features

  • Live 911 Emergency Reports - Displays real-time emergency calls from Seattle 911 Dispatch.
  • Crime Data Visualization - Historical crime reports from SPD data (2024-Present).
  • Safe Route Navigation - Avoids high-crime areas using dynamic rerouting.
  • Interactive UI - Users can toggle different data layers.

Technologies & Libraries Used

  • Mapbox GL JS - For interactive mapping.
  • OpenStreetMap (OSM) - Navigation and infrastructure data.
  • Seattle Open Data API - Live crime and emergency data.
  • Python (Pandas, Geopandas) - Data processing.
  • Turf.js - Geospatial analysis.
