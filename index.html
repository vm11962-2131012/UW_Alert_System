U District Crime/Incident Alert Map


About the Map

This interactive map is designed to help students and faculty, residents, and visitors stay informed about crime incidents in the University District. Using real-time data from the Seattle Open Data Portal, this map provides recent police and crime reports, citizen-reported incidents, and EMS calls.

Key Features

Live Crime Updates

A live-updating map of all police reports, crime reports, EMS calls or citizen-reported incidents


Rerouting for Safe Navigation

Rerouting function for users to their destinations on campus, avoiding any active crime scenes (buffers around certain areas), with adjustable travel methods (isochrone mapping)


Crime Data in UW

Seattle Police Department crime data from all of 2025


Interactive Crime Alerts

Popup alerts of crime incidents with information about the incident, including date, time, and location
